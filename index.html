Welcome to Work Flow!

This application was designed around replacing a paper work order system I used when working for a Fire Suppression/Life Safety company. Here's how it works:

  1. First, you create a work order and fill in all of the applicable information. The new work order is passed on to the admin section where it waits for a "job number" or labour code.
  2. In the admin section, the admin team would generate a labour code in their ERP software and enter it in the field provided. After a job number has been added to the work order, it will be automatically be moved to the "Service" section of the application.
  3. A technician would go into the Service section, select a job from the list, and fill out their part of the work order including their notes and all parts used on that particular job.
  4. Once the job is completed, it gets moved to the History section where the entire contents of the work order will be viewable.
